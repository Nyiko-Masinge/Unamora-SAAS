# UI Wireframes

## Core Screens
- landing page and marketing entry
- search and discovery results
- professional profile details
- booking flow and confirmation
- client dashboard
- tradesperson dashboard
- admin operations workspace

## Interaction Notes
- the landing page should quickly communicate trust and value
- search results should make relevance obvious
- booking flows should reduce uncertainty and highlight next actions
- dashboards should expose immediate status and decision points

## Design Direction
The UI should feel modern, dependable, and practical, with a strong emphasis on trust signals and low-friction task completion.
